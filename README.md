## Problem Description
Colleges and organizations often conduct multiple events such as workshops, cultural 
programs, and seminars. Managing registrations manually leads to duplicate entries, missed 
communications, and poor coordination. \
An Event Management System centralizes event creation, registration, and participant tracking

## How to Run the Code
1. Download the EventManager package from /src and move it to the src folder of your eclipse workspace.
2. Download the .properties and .db files and move them to the folder containing your src folder.
3. Make a new folder inside the EventManager package using eclipse and name it lib.
4. Download the mapDB JAR file from /lib and drag and drop it to the newly created folder in eclipse.
5. Right click on the EventManager package and select properties.
6. Go to Java Build Path on the left menu and go to libraries.
7. Click on Classpath,Add JARs from the right menu, then select the lib folder of our package and click on the mapDB jar file.
8. To populate the databases, run Setup.java
9. To run the Event Management System, run EventManagementSystem.java
